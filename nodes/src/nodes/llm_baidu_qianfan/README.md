# Baidu Qianfan

Use the Baidu Qianfan node to call ERNIE chat models through Qianfan's OpenAI-compatible API.

## Configuration

- **Model**: ERNIE model name, such as `ernie-4.5-turbo-128k`.
- **API key**: Qianfan API key with access to the selected model.
- **Base URL**: OpenAI-compatible Qianfan endpoint. The default is `https://qianfan.baidubce.com/v2`.

For Baidu's international Qianfan endpoint, set the base URL to `https://api.baiduqianfan.ai/v1`.
