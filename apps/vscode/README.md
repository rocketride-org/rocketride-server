# RocketRide

Build, run, and scale AI pipelines — without leaving your IDE.

RocketRide gives you a visual pipeline builder, 50+ ready-to-use nodes, multi-agent orchestration, and a blazing-fast engine. Connect LLMs, vector databases, and data tools into production-ready workflows in minutes. No browser. No boilerplate. Just ship.

## Why RocketRide

- **Visual pipeline builder** — Drag, drop, and wire up powerful AI workflows right inside VS Code.
- **50+ nodes out of the box** — 13 LLM providers, 8 vector databases, OCR, NER, PII anonymization, and more. Ready to go.
- **Multi-agent orchestration** — Scale agents with built-in CrewAI and LangChain support.
- **Built for production** — High-performance C++ engine. Deploy on Docker, on-prem, or RocketRide Cloud (coming soon).
- **SDKs for TypeScript, Python & MCP** — Embed pipelines in your apps or expose them as tools for AI assistants.

## Quick Start

1. Click the RocketRide icon in the Activity Bar
2. Choose how to run the server:
   - **Local (Recommended)** — One click. No setup.
   - **On-Premises** — Full control on your own hardware.
   - **RocketRide Cloud** (coming soon) — Managed hosting. Zero infrastructure.

## Build a Pipeline

Create a `*.pipe` file and the extension opens it in the visual builder canvas.

1. Start with a source node: **webhook**, **chat**, or **dropper**
2. Wire up nodes by connecting input and output lanes
3. Hit play to run — or launch from the **Connection Manager**
4. Monitor running pipelines with real-time analytics — trace calls, token usage, memory, and more
